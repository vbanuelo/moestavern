## Color Palette

Moe's Tavern is neon and dirty, but full of character. Our color selections intend to reflect this:

- Main text color: #ffffff
- Body background color: rgba(4, 170, 109, 0.5)
- Header border color: #375e97
- Blocks scheme:
  - Text color: #ffffff
  - Background color: #f9f9f9
  - Dropshadow: 0 2px 5px rgba(0, 0, 0, 0.75)
  - See live CSS for sizing
- Card scheme:
  - Text color: #ffffff
  - Background color: rgba(135, 250, 90, 0.5)
  - Border: 1px rgba(135, 250, 90, 0.5)
- Hover
  - See CSS for dimensions
  - Dropshadow: 0 15px 30px rgba(255, 23, 68, 0.75)
  - Background: rgba(43, 255, 61, 0.75)
  - Text color \(on hover\): rgb(0, 0, 0)
- Visited links
  - Shadow: #375e97
  - Background: rgba(135, 250, 90, 0.5)
  - Color: rgb(235, 241, 236)
